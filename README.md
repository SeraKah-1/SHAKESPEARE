# Wordsmith: Mesin Penulis Cerita Cyberpunk

Halo! Ini adalah proyek kecil untuk kamu yang suka nulis cerita, tapi kadang macet di tengah jalan. Wordsmith bantu kamu ngerakit plot, nulis draf, dan jagain supaya cerita kamu nggak melantur ke mana-mana.

## Apa yang Bisa Dilakukan?

- **Rancang Plot**: Bantu kamu bikin garis besar cerita supaya nggak bingung mau nulis apa selanjutnya.
- **Tulis Draf**: Ngerakit kata-kata jadi paragraf novel yang enak dibaca (bukan kayak naskah drama atau list belanjaan).
- **Koreksi Otomatis**: Ada "editor" galak yang bakal ngecek kalau tulisan kamu melenceng dari aturan dunia yang udah kamu buat.
- **Simpan Memori**: Dia inget siapa aja karakter kamu dan apa aja yang udah terjadi, jadi nggak bakal ada karakter yang tiba-tiba idup lagi padahal udah mati di bab sebelumnya.

## Cara Pakai

1. **Pasang Bahan**: Kamu butuh kunci API (Gemini atau Groq) dan database Supabase kalau mau simpan ceritanya permanen.
2. **Nyalakan**: Jalankan `npm install` terus `npm run dev`.
3. **Mulai Nulis**: Buka di browser, bikin cerita baru, dan kasih ide awal kamu. Sisanya biar Wordsmith yang bantu olah.
